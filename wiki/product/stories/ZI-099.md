### Scenario: Direct fix for reported issue

### Scenario: Fix holds for similar configurations

- **When** they perform the same action
- **Then** the fix still applies and the action succeeds
- **And** no edge case triggers the original failure

### Scenario: Failure is self-diagnosable

- **Then** the error message identifies the specific field, carrier, or configuration causing the problem
- **And** the merchant can resolve it without opening a support ticket
