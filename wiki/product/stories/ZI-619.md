## User Story

As a **B2B wholesale merchant on the Popular plan shipping ~233 international orders via FedEx and UPS from France**,
I want **the Tax ID that Shopify already carries in each order's customer notes to flow automatically into the carrier label request**,
so that **I stop burning 2-3 minutes per shipment on manual copy-paste and eliminate the customs clearance rejections that happen when a VAT number is mistyped or forgotten**.

---

## Acceptance Criteria (Simple)

- [ ] When generating a label, the app reads the customer Tax ID from the Shopify order's customer note field and auto-populates it into both shipping and billing TAX ID address fields.
- [ ] The auto-populated Tax ID is included in the carrier API request payload sent to FedEx and UPS.
- [ ] If no Tax ID is present in the customer notes, the TAX ID field remains empty and editable (no errors, no blocking).
- [ ] Merchant Wako Chapurlat (majo-garn.myshopify.com) can generate a label for an order with a customer-profile Tax ID and see it pre-filled without manual entry.
- [ ] The merchant can override the auto-populated Tax ID before label generation if needed.

---

## Acceptance Criteria (Given/When/Then)

### Scenario: Tax ID auto-populates from Shopify customer notes
- **Given** an order exists where the Shopify customer profile contains a Tax ID (VAT number) surfaced in the order's Customer Notes section
- **When** the merchant opens the order in MCSL and initiates label generation
- **Then** the TAX ID fields in both shipping and billing address sections are pre-filled with the customer's Tax ID value

### Scenario: No Tax ID present in customer notes
- **Given** an order exists where the Shopify customer profile has no Tax ID and no relevant data appears in the Customer Notes
- **When** the merchant opens the order in MCSL
- **Then** the TAX ID fields remain empty, editable, and label generation proceeds without error

### Scenario: Tax ID passed to carrier API for customs clearance
- **Given** a label is being generated for an international shipment with an auto-populated Tax ID
- **When** the carrier API request is submitted to FedEx or UPS
- **Then** the Tax ID value is included in the appropriate tax identification field of the carrier request payload and the generated label/customs documents reflect the Tax ID
