## User Story

As a **wine retailer managing hundreds of product variants where each legally requires Adult Signature configuration**,
I want **the CSV product import/update to correctly apply signature settings to all products in bulk**,
so that **when I add new wine vintages daily I can use self-serve CSV import to configure Adult Signature for 50+ products at once instead of opening a support ticket for manual one-by-one updates that take a full business day and block my ability to ship those orders**.

## Acceptance Criteria (Simple)

- [ ] CSV product import with signature option column correctly applies Adult Signature to all imported products
- [ ] CSV product update with signature option column correctly updates existing products with new signature settings
- [ ] Pierre (grapebottle.myshopify.com) can bulk-configure Adult Signature for new wine products via CSV without support intervention
- [ ] CSV signature import works correctly for merchants using FedEx REST API (post-migration)
- [ ] CSV import logs/validation clearly indicates when signature option fails to apply and why

## Acceptance Criteria (Given/When/Then)

### Scenario: CSV import applies signature settings correctly (Happy path)
- **Given** a Shopify merchant has completed FedEx REST API migration
- **And** a CSV file includes product rows with signature option set to "Adult Signature Required"
- **When** the merchant imports the CSV via product import tool
- **Then** all imported products are configured with Adult Signature at product level
- **And** labels generated for those products show "Adult Signature Required" on the label

### Scenario: CSV update modifies signature settings for existing products (Broader fix)
- **Given** existing products are configured with no signature or different signature settings
- **And** a CSV update file specifies new signature options for those products
- **When** the merchant imports the CSV update
- **Then** product signature settings are updated to match the CSV values
- **And** subsequent labels reflect the new signature configuration

### Scenario: CSV import validation catches signature configuration errors (Prevention)
- **Given** a merchant attempts CSV product import with signature options
- **When** the import process completes
- **Then** import results clearly indicate which products successfully configured signature settings
- **And** any failures to apply signature settings are logged with specific error messages
- **And** merchant can see validation summary showing signature column processing status
