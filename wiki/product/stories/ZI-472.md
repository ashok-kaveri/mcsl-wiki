## User Story

As a **WooCommerce merchant who ships media products (books, CDs, DVDs) and is eligible for USPS Media Mail rates**,
I want **USPS Media Mail to appear as a shipping option at checkout without requiring a dev enhancement or 16-month wait**,
so that **I can offer my customers the lowest-cost USPS option for media shipments (often 50% cheaper than Priority Mail), instead of either overpaying for Priority Mail on every order or abandoning the app to process Media Mail labels manually through USPS directly, which defeats the purpose of paying for a shipping automation plugin**.

---

## Acceptance Criteria (Simple)

- [ ] USPS Media Mail service appears in WooCommerce Shipping Services automation rules configuration
- [ ] When automation rules are configured to show Media Mail, it displays at checkout for eligible products
- [ ] Media Mail rates are fetched correctly from USPS API (XM service code) and displayed with accurate pricing
- [ ] Customer Kenny's store (ticket #302656) can see and select Media Mail for their media product orders
- [ ] Plugin documentation updated to explain Media Mail eligibility requirements (product type restrictions per USPS rules)

---

## Acceptance Criteria (Given/When/Then)

### Scenario: Customer configures Media Mail in automation rules
- **Given** a WooCommerce store is connected with USPS carrier in the plugin
- **And** the store ships media products (books, CDs, DVDs, educational materials)
- **When** the merchant navigates to automation rules for USPS
- **Then** USPS Media Mail appears as an available service option in the dropdown/checkbox list
- **And** the merchant can enable/disable Media Mail just like Priority Mail, First Class, etc.

### Scenario: Media Mail rates display at checkout
- **Given** Media Mail is enabled in the store's USPS automation rules
- **And** a customer adds a media product to their cart
- **When** the customer reaches checkout and enters a US shipping address
- **Then** USPS Media Mail appears as a shipping option with accurate rate fetched from USPS API
- **And** the rate is significantly lower than Priority Mail (typical 40-60% cheaper for books)
- **And** selecting Media Mail allows order completion successfully

### Scenario: Non-media products correctly exclude Media Mail
- **Given** Media Mail is enabled in USPS automation rules
- **And** a customer adds a non-media product (clothing, electronics, tools, etc.) to their cart
- **When** the customer reaches checkout
- **Then** Media Mail does NOT appear as an option (USPS restricts Media Mail to books, sound recordings, video recordings, educational materials only)
- **And** only eligible USPS services (Priority, First Class, etc.) are shown
- **And** this behavior follows USPS eligibility rules without requiring merchant configuration per-product
