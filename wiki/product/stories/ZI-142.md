# Ticket #304193 — Order Modification: Notes & Name Change Sync

- **Customer**: Zulay Dominguez (unknown store)
- **Duration**: 2024-12-16 to 2025-08-14 (ongoing)

## Timeline & Key Phases

### Phase 1: Feature Request — Order Note Sync (2024-12-16)
- Customer suggested enhancement: trigger the "order being modified" alert when a note is edited or added to an order, similar to the existing alert for address/product changes
- Submitted from the app's orders page
- Request forwarded to product team (Comment #2)

### Phase 2: Name Change Enhancement Under Review (2025-01-23)
- Internal testing of order modification feature for first/last name changes on test store gamingzonexyz.myshopify.com — notification not appearing after 5 minutes (Comment #4)
- Senior team noted enhancement would be considered for future releases (Comment #5)
- Customer acknowledged and expressed interest (Comment #7)

### Phase 3: Follow-up & Confirmation (2025-04-10)
- Support followed up; customer confirmed still interested in the enhancement (Comment #9)

### Phase 4: Name Change Enhancement Deployed (2025-08-13)
- App enhanced to fetch modifications in first name and last name; feature enabled for the customer's store (Comment #12)
- Customer notified via email after failed phone call attempt (Comment #13, #14)

### Phase 5: Order Note on Packing Slip Issue (2025-08-14)
- Customer raised related issue: order notes added after import do not appear on packing slips (Comment #17)
- Support clarified that FedEx and Stamps USPS carriers do not support order notes on labels (Comment #16)
- Customer clarified the issue is about packing slips, not labels (Comment #17)
- Senior team confirmed order note update via order modification feature will be enhanced in upcoming release (Comment #20)

## Open Issues

1. **Order notes not synced after import** — Notes added to Shopify orders after import to the app do not appear on packing slips. Area: order-management. (Comment #20)

## Resolved Issues
