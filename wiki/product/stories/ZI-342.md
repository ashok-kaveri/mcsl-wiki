## User Story

As a **warehouse manager for a Shopify development agency building client stores with multiple box templates configured for different product types (small jewelry boxes, medium apparel boxes, large bulk boxes)**,
I want **the box template name or ID printed directly on the shipping label PDF (e.g., "Box Type: Medium Apparel" or "Pkg ID: BOX-02") in a visible location near the barcode**,
so that **my warehouse team knows exactly which physical box to grab from the shelf without opening Shopify to cross-reference the order, reducing pick-pack time from 5 minutes to under 2 minutes per order, eliminating packing errors that cause carrier dimensional weight billing disputes or damaged shipments, and allowing us to onboard new warehouse staff without requiring them to memorize box selection rules**.

---

## Acceptance Criteria (Simple)

- [ ] Box template name displays on shipping label PDF in a dedicated field (e.g., near package dimensions or barcode)
- [ ] Merchant can choose label position for box identifier: top-right corner, below barcode, or near package dimensions
- [ ] Box identifier is large enough to read at arm's length (minimum 10pt font)
- [ ] Multi-package shipments show correct box template per label (Package 1: "Large Box", Package 2: "Small Box")
- [ ] Setting can be toggled: "Show box template on label PDF" (default: disabled for backward compatibility)

---

## Acceptance Criteria (Given/When/Then)

### Scenario: Merchant enables box template display on labels
- **Given** merchant has configured multiple box templates (Small, Medium, Large)
- **And** merchant enables "Show box template on label PDF" setting
- **When** merchant generates a shipping label for an order packed in "Medium Apparel Box"
- **Then** label PDF displays "Box: Medium Apparel" in the configured position (e.g., top-right corner)
- **And** text is clearly visible at arm's length (minimum 10pt font)

### Scenario: Multi-package shipment with different boxes
- **Given** order requires 2 packages: 1 in "Large Bulk Box", 1 in "Small Jewelry Box"
- **When** merchant generates labels for the multi-package shipment
- **Then** Package 1 label shows "Box: Large Bulk"
- **And** Package 2 label shows "Box: Small Jewelry"
- **And** warehouse team can identify correct boxes without cross-referencing Shopify

### Scenario: Merchant customizes box identifier position
- **Given** merchant prefers box identifier near package dimensions (not top-right)
- **When** merchant navigates to Settings → Label Customization → Box Identifier Position
- **Then** merchant can select from: "Top-right corner", "Below barcode", "Near package dimensions", "Custom coordinates"
- **And** test label preview shows box identifier in selected position
