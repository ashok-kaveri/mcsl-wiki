# Ticket #354696 — Order Report CSV Export & Missing City Column

- **Customer**: Zac (agro-coffee.myshopify.com)
- **Duration**: 2025-10-16 to 2025-10-22 (ongoing)

## Timeline & Key Phases

### Phase 1: CSV Export Request (2025-10-16 to 2025-10-17)
- Customer asked for a way to export ~1,000 orders into CSV to analyze shipping rates and free shipping thresholds across Canada
- Support initially offered to generate a report manually (Comment #3)
- Senior team enabled the Order Report export feature for the store (Comment #7)
- Customer shown how to use Export > Select Date Range > Provide Email > Export Orders (Comment #8)

### Phase 2: Missing City Field (2025-10-20)
- Customer confirmed the export works well; province, zip code, and street address are included
- Requested addition of Ship To Address City/Town to help assess service to remote locations in BC (Comment #9)
- Support confirmed City is not currently available in the report; only Street Address, State, and Postal Codes are included (Comment #10, #11)
- Escalated to L3 via Slack (Comment #11)

### Phase 3: Enhancement Confirmed (2025-10-22)
- Senior team confirmed Ship To Address City will be added to the exported order report in the upcoming release (Comment #13)
- Customer acknowledged and expressed satisfaction; noted they used Google Sheets formulas with a postal code database as a workaround in the meantime (Comment #15)

## Open Issues

1. **Ship To Address City missing from Order Report** — The exported CSV order report does not include the City column. Only Street Address, State, and Postal Code are present. Enhancement confirmed for upcoming release. Blocked: dev team. Severity: low-moderate, has a workaround via postal code lookup. Area: order-management. (Comment #13)

## Resolved Issues

1. **Order Report export feature** — Feature enabled for the store; customer can now export orders as CSV with date range selection. (Comment #7, #8)
