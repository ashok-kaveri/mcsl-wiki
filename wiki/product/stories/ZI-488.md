## Open Issues

1. **Adult Signature automation rule not working post-REST migration** — Automation rule signature option fails to apply ASR to labels after FedEx REST API migration. Labels generate with DSR (Direct Signature) instead of ASR (Adult Signature Required) when order total exceeds $100. Blocked: customer (daily operations impacted, cannot use automation rules for ASR). Severity: high (business-critical, wine retailer requires ASR for all shipments). Area: `carrier-migration`. (Comment #46, #79, #80)

2. **CSV signature update feature broken post-REST migration** — Product CSV import/update with signature option not applying Adult Signature to products. Customer attempted bulk CSV update but ASR not reflected on products. Blocked: customer (cannot use CSV for bulk ASR configuration). Severity: high (eliminates self-serve bulk update workaround). Area: `product-management`. (Comment #44, #46)

3. **Daily manual product ASR updates required** — Customer adds new wine products daily (new vintages replace old), each requiring manual ASR configuration by support team. Both automation rules AND CSV bulk updates are broken, leaving only manual one-by-one updates. Blocked: customer (ongoing manual dependency, no self-serve path). Severity: high (operational burden, scalability issue, blocks business growth). Area: `product-management`. (Comment #81, #82)

4. **Packaging type validation errors on label generation** — "Packaging type not allowed" error when generating labels with custom packaging and valid services. FedEx reports "One Rate" special service incorrectly passed in request. Blocked: engineering (code fix required). Severity: medium (intermittent label failures). Area: `label-generation`. (Comment #76, #79)

## User Story

As a **wine retailer paying $9/month for shipping automation whose business model requires adding new wine vintages daily**,
I want **any self-serve path to configure Adult Signature for new products** (automation rules, CSV import, or bulk product editor),
so that **I don't have to open support tickets every single day requesting manual product updates that delay my ability to ship orders, waste support team capacity on repetitive manual work, and make me question whether a $9/month app that requires daily support intervention is sustainable for my business**.
