## Timeline & Key Phases

### Phase 1: Initial Setup & Configuration (2026-05-23 to 2026-05-25)
- Customer configured rate automation for CounterCool product with two shipping zones:
  - Zone 1: Contiguous US + AK + HI → $15 flat rate per unit
  - Zone 2: PR + Guam + Marshall Islands + AA → $20 flat rate per unit
- Tested on production site, reverted to avoid customer-facing errors
- Reported that checkout shows "no shipping available" error for PR, Guam, Marshall Islands, AA

### Phase 2: Support Troubleshooting (2026-05-23 to 2026-05-25)
- Support added PR as "US-Puerto Rico" option in shipping zone UI → rates appeared for PR
- Support added AA as "US-AA" option in shipping zone UI → rates appeared for AA
- **Gap identified**: No "US-Guam" or "US-Marshall Islands" options available in shipping zone dropdown
- Support suggested adding Guam/Marshall Islands to existing PR zone, but customer rejected (would show wrong $15 rate)

### Phase 3: L3 Investigation & Root Cause (2026-05-25 to 2026-05-27)
- L3 (Veena) granted collaborator access to investigate
- Support removed and re-added MCSL app rates from shipping profile
- L3 captured request log for Guam address check (ID: aa7bb900c0458a02804875b0b83872e4)
- **Root cause identified** (Comment #17): Automation rule is not triggering for Guam addresses. Automation summary in request log is blank.
- L3 decision: "This needs a fix" — scheduled for next cycle. Recommended workaround: use Shopify's native flat rate for Guam/Marshall Islands.

### Phase 4: Workaround Deployed (2026-05-27)
- Customer disabled $20 flat rate automation rule
- Removed special shipping zones from $15 rule
- Currently offering $15 rate to all destinations as interim solution
- Awaiting fix in next development cycle (approx. 2 weeks)
