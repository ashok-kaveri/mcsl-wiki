## Cross-Theme Clusters Worth Backlog Attention

- **Floating-point rounding in customs value passed to FedEx** — ZI-560. Customs value sent as `127.39999999999999` while declared value is `127.40`, causing FedEx to reject. This is a textbook JS float artifact (likely multiplying/summing line-item values before rounding). One-line fix at the customs-value serialisation boundary (round to 2 decimals) — but worth auditing all carrier payloads (FedEx, DHL, others) for the same pattern across customs/declared/duty fields.
- **WSS install blocker: WordPress capability check on plugin entry page** — ZI-562. "Let's Start Fulfilling" throws `Sorry, you are not allowed to access this page` for an Admin role user (reproduced in Incognito; other plugins open fine). This is a hard onboarding wall — customer cannot configure carriers at all. Likely a missing/over-restrictive `current_user_can()` check or a capability mapping bug introduced by a recent WSS release. Should be triaged urgently against the WSS plugin's recent changelog.
