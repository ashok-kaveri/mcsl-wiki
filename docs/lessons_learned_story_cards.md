# Standing Rules — Bulk Card Generation

## Rule 1 — Never use card 1 as a template
Generate card 1, then stop. Do not proceed to card 2 until card 1 has been explicitly validated.

## Rule 2 — Validate card 1 against the quality checklist before continuing
Check every item on the checklist. If any item fails, rewrite and re-validate before moving on.

## Rule 3 — Each card must have specific, non-generic AC
Acceptance criteria must be written for that card's scenario. Generic or copy-pasted AC is a failure.

## Rule 4 — Each card gets a unique scenario structure
Do not copy-paste structure across cards. Vary context, actors, and conditions for each one.

## Rule 5 — Apply stop → rewrite → re-validate on every failure
If any card fails validation, stop the batch. Rewrite that card. Re-validate it. Only then continue.
